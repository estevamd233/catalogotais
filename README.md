# Portfólio - Thaís Alves

Site estático de portfólio profissional criado para apresentar serviços de manicure, galeria e contato.

Como rodar localmente:

```bash
# Entre na pasta do projeto
cd "c:/Users/PC0345/Desktop/Serviços Freelancer/Catálogo Taís"
# Inicie um servidor HTTP simples (Python 3)
python -m http.server 8000
# Abra http://localhost:8000 no navegador
```

Arquivos principais:

- `index.html` - página principal (sobre, serviços, galeria, contato)
- `styles.css` - estilos
- `F1C.JPG` - imagem usada como capa/galeria
